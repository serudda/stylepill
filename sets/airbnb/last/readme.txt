Airbnb Theme
============

Set #3: Airbnb main subnavs
---------------------------

This set contains:
- Simple navbar: ma-subnav--simple
- Tabs navbar: ma-subnav--tabs

Use:
You have to use the class: 'ma-subnav' to get base button styles, and add the respective modifier.

You can find the list of html components on 'subnavs.html' file.
